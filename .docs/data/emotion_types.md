# Minder 감정 분류 체계 (총 48개)

Minder 앱에서 사용하는 감정 분류는 로버트 플루치크(Robert Plutchik)의 감정 휠 이론을 기반으로 하며, 총 48개의 감정 상태를 정의합니다.

## 1. 기본 감정 계열 (Detailed Emotions) - 24개
각 기본 감정(8가지)은 강도에 따라 **강함(Level 3)**, **중간(Level 2)**, **약함(Level 1)**으로 세분화됩니다.

| 계열 (Base) | 강함 (Strong, Lv 3) | 중간 (Medium, Lv 2) | 약함 (Weak, Lv 1) |
|---|---|---|---|
| **기쁨 (JOY)** | 황홀 (Ecstasy) | 기쁨 (Joy) | 평온 (Serenity) |
| **신뢰 (TRUST)** | 감탄/존경 (Admiration) | 신뢰 (Trust) | 용인 (Acceptance) |
| **두려움 (FEAR)** | 공포 (Terror) | 두려움 (Fear) | 우려 (Apprehension) |
| **놀람 (SURPRISE)** | 경악 (Amazement) | 놀람 (Surprise) | 산만/주의분산 (Distraction) |
| **슬픔 (SADNESS)** | 비탄 (Grief) | 슬픔 (Sadness) | 수심 (Pensiveness) |
| **혐오 (DISGUST)** | 증오 (Loathing) | 혐오 (Disgust) | 지루함 (Boredom) |
| **분노 (ANGER)** | 격노 (Rage) | 분노 (Anger) | 짜증 (Annoyance) |
| **기대 (ANTICIPATION)** | 경계 (Vigilance) | 기대 (Anticipation) | 관심 (Interest) |

## 2. 복합 감정 (Complex Emotions, Dyads) - 24개
두 가지 기본 감정이 결합하여 형성되는 감정입니다. 결합하는 두 감정 사이의 거리에 따라 1차, 2차, 3차로 나뉩니다.

### 1차 이중감정 (Primary Dyads) - 8개
*(인접한 두 감정의 결합, 거리 1)*

1.  **사랑 (Love)**: 기쁨 + 신뢰
2.  **굴복/순종 (Submission)**: 신뢰 + 두려움
3.  **경외 (Awe)**: 두려움 + 놀람
4.  **못마땅함 (Disapproval)**: 놀람 + 슬픔
5.  **후회 (Remorse)**: 슬픔 + 혐오
6.  **경멸 (Contempt)**: 혐오 + 분노
7.  **공격성 (Aggressiveness)**: 분노 + 기대
8.  **낙관 (Optimism)**: 기대 + 기쁨

### 2차 이중감정 (Secondary Dyads) - 8개
*(하나 건너뛴 두 감정의 결합, 거리 2)*

9.  **죄책감 (Guilt)**: 기쁨 + 두려움
10. **호기심 (Curiosity)**: 신뢰 + 놀람
11. **절망 (Despair)**: 두려움 + 슬픔
12. **불신 (Unbelief)**: 놀람 + 혐오
13. **부러움/시기 (Envy)**: 슬픔 + 분노
14. **냉소 (Cynicism)**: 혐오 + 기대
15. **자부심 (Pride)**: 분노 + 기쁨
16. **숙명/운명론 (Fatalism)**: 기대 + 신뢰

### 3차 이중감정 (Tertiary Dyads) - 8개
*(두 개 건너뛴 두 감정의 결합, 거리 3)*

17. **환희 (Delight)**: 기쁨 + 놀람
18. **감상 (Sentimentality)**: 신뢰 + 슬픔
19. **수치심 (Shame)**: 두려움 + 혐오
20. **격분 (Outrage)**: 놀람 + 분노
21. **비관 (Pessimism)**: 슬픔 + 기대
22. **병적 집착 (Morbidness)**: 혐오 + 기쁨
23. **지배 (Dominance)**: 분노 + 신뢰
24. **불안 (Anxiety)**: 기대 + 두려움

### 반대 감정의 결합 (Opposite Emotions / Conflict) - 4개
*(정반대에 위치한 두 감정의 결합, 거리 4)*

25. **씁쓸함 (Bittersweetness)**: 기쁨 + 슬픔
26. **애증 (Ambivalence)**: 신뢰 + 혐오
27. **얼어붙음 (Frozenness)**: 두려움 + 분노
28. **혼란스러움 (Confusion)**: 놀람 + 기대
